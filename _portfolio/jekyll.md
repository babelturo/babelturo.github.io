루비로 로컬 환경에서 테스트 후, GitHub Desktop을 사용해서 업로드. Git에 익숙하지 않은 사람에게는 이게 편함.

# Ruby

명령창(command prompt)에서 블로그 페이지로 이동한 다음 아래와 같이 입력하면...

```powershell
jekyll serve
```

화면에 표시되는 IP주소를 웹브라우저에 입력하여 블로그를 미리 확인할 수 있다. 수정할 때마다 깃에 업로드를 하는 번거로움을 줄여준다.
